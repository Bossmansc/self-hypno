# How to Get Your APK

## Option 1: Cloud Build (No Android Studio Required)
1. Commit this code to a **GitHub Repository**.
2. Go to the **Actions** tab in your GitHub repository.
3. You will see a workflow named "Build Android APK" running.
4. When it finishes (green checkmark), click on it.
5. Scroll down to the **Artifacts** section and download `app-debug.apk`.
6. Send this file to your phone and install it.
